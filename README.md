# WordPress theme xb-wp-theme

WordPress theme build with [jQuery](https://jquery.com/) as JavaScript library
and [Skeleton](http://getskeleton.com/) as responsive CSS boilerplate.

### Setup

Install requirements
* `npm install`
* `bower install`

Run
* `gulp`
